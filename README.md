# Colorful-Geometric-Forms

Um projeto de estudo sobre transições e animações CSS, apresentando formas geométricas coloridas que sofrem modificações visuais através de efeitos interativos.

## Descrição

Este repositório contém uma coleção de animações CSS puras, sem JavaScript, focadas em transições e animações de elementos geométricos. Cada pasta representa uma animação diferente, demonstrando técnicas variadas de CSS para criar efeitos visuais dinâmicos.

## Estrutura do Projeto

- `index.html` - Página principal com links para todas as animações
- `balls-animation/` - Animação de bolas
- `balls-animation-two/` - Segunda variação da animação de bolas
- `balls-transition/` - Transição de bolas
- `rectangular-line-animation/` - Animação de linhas retangulares
- `rectangular-line-transition/` - Transição de linhas retangulares
- `style.css` - Estilos globais
- `global.css` - Estilos adicionais

## Como Executar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador web
3. Navegue pelas diferentes animações clicando nos links

## Hospedagem

O projeto está hospedado em: https://colorful-geometric-forms.vercel.app/

## Tecnologias Utilizadas

- HTML5
- CSS3 (Transições e Animações)
